JavaScript

Ce este JavaScript?

JavaScript este un limbaj de programare puternic, rapid si flexibil care îți permite să implementezi lucruri complexe pe paginile web — de fiecare dată când o pagină web face mai mult decât să afișeze informații statice pentru ca tu să te uiți — afișând actualizări de conținut în timp util sau hărți interactive, grafice animate 2D/3D, video jukebox, etc. — probabil că JavaScript este implicat.

Istorie

S-ar putea să fi văzut termenul "ES6" sau "Javascript ES6" și sa va întrebati ce înseamnă de fapt. Sa facem o mica istorie. JavaScript a fost introdus în anul 1995 de compania Netscape Communications ca limbaj de scripting pentru web designeri și programatori pentru a interacționa cu paginile web. În anul următor, Netscape a trimis JavaScript la o organizație de dezvoltare a standardelor numită Ecma International pentru a crea standarde pentru un limbaj de scripting (un tip de limbaj de programare). În 1997, Ecma International a lansat ECMA-262 care stabilește standarde pentru prima versiune a limbajului de scripting numit ECMAScript, prescurtat ES.
De asemenea, s-ar putea să vedeți ES2015 în loc de ES6, dar ambele terminologii se referă la a sasea ediție a ECMAScript, care a fost lansată în 2015.

Diferentele dintre ES5 si ES6

Chiar daca exista si ES7 sau ES8, ES6 este cea mai mare actualizare făcută de ECMAScript de la prima ediție lansată în 1997! Unii dezvoltatori supranumesc ES6 ca "JavaScript modern" din cauza tuturor adăugărilor majore. Au fost adăugate atâtea caracteristici extraordinare pentru a ajuta dezvoltatorii JavaScript care includ:

  • noi cuvinte cheie precum: let si const pentru a declara variabile,
  • noua funcție de sintaxă utilizând funcțiile cu săgeți (arrow functions),
  • crearea de clase,
  • parametrii cu valori implicite,
  • si multe altele.

Browserele suporta acum majoritatea caracteristicilor ES6 care permit dezvoltatorilor să profite de aceste noi adăugari. ES6 permite programatorilor să economisească timp și să scrie coduri mai concise. Luați, de exemplu, sintaxa ES5 pentru expresii de funcții:

var greeting = function() {
	console.log('Hello World!');
};

Cu funcțiile de săgeată din ES6, putem transforma expresia de mai sus în:

const greeting = () => console.log('Hello World');
